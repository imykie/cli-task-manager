# Task CLI
Task CLI is a simple command line task manager that can be used to manage your day to day tasks

### Installation
Change directory to the root of the project
```shell
go install .
```

### Usage
To add task
```shell
task add "Do Something"
```
List Tasks
```shell
task list
```
Do Task
```shell
task do 1 2
```
Update Task
```shell
task update 1 "Change task"
```
Get help
```shell
task help
```
